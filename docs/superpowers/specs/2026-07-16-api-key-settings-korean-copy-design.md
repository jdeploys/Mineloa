# API 키 설정 한국어 문구 설계

## 범위

설정 화면의 `ApiKeySettings` 카드에서 사용자에게 노출되는 영어 문구를 한국어로 통일한다. `OpenAI`와 `API`는 고유 표기로 유지한다. 저장·삭제·상태 갱신 로직, 비밀 키 비노출 보장, 다른 화면의 문구는 변경하지 않는다.

## 문구

- 제목: `API 키 설정`
- 상태: `상태 확인 불가`, `설정됨`, `설정되지 않음`
- 검증 시각: `마지막 검증:`
- 필드/버튼: `OpenAI API 키`, `API 키 저장`, `API 키 삭제`
- 오류: 상태 조회, 저장 후 상태 갱신, 키 검증, 삭제 후 상태 갱신, 삭제 실패를 각각 자연스러운 한국어 문장으로 표시한다.

## 검증

컴포넌트 테스트에서 한국어 제목·상태·필드·버튼을 확인한다. 가장 가까운 비변경 동작으로 저장된 키가 화면과 Desktop API에 노출되지 않는 기존 보안 회귀 검사를 유지한다. 문서용 설정 화면을 다시 캡처해 실제 렌더링 픽셀에 영어 UI 문구가 남지 않았는지 확인한다.
